# Image Placeholders

This folder should contain the following images:

1. **people.jpg** - Photo of happy cooking students or people enjoying food
   - Recommended size: 800x600px
   - Used on: Home page

2. **fruit.jpg** - Photo of fresh fruits or cooking ingredients
   - Recommended size: 800x600px
   - Used on: Courses page

3. **graph.png** - Chart or infographic about cooking/courses
   - Recommended size: 800x600px
   - Used on: Contact page, Course details, Thank you page

4. **cat.jpg** - Cute cat photo (for error page)
   - Recommended size: 800x600px
   - Used on: Error page

5. **icon.png** - Small icon/logo for the website
   - Recommended size: 64x64px
   - Used as: Favicon

## Where to get free images:
- Unsplash.com
- Pexels.com
- Pixabay.com

## Or use placeholder services:
Replace the image src in your EJS files with:
- https://placehold.co/800x600?text=People
- https://placehold.co/800x600?text=Fruits
- https://placehold.co/800x600?text=Graph
- https://placehold.co/800x600?text=Cat
- https://placehold.co/64x64?text=Icon
